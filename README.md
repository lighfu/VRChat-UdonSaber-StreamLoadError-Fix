# VRChat-UdonSaber-StreamLoadError-Fix
VRChatのうどんセーバーにおけるStreamLoadError ([AVProVideo] Error: Loading failed.  File not found, codec not supported, video resolution too high or insufficient system resources.)を、可能な限り修正します。

使い方は [こちら](https://ajisaiflow.booth.pm/items/7673438) をご覧ください。
